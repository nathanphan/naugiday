# Quickstart: Ingredient Management

## Prerequisites

- Flutter stable with Dart 3.10.1
- Xcode and iOS simulator/device

## Setup

```bash
flutter pub get
dart run build_runner build --delete-conflicting-outputs
```

## Run (iOS)

```bash
flutter run -d ios
```

## Tests

```bash
flutter test
```

## Useful Commands

```bash
dart run build_runner watch --delete-conflicting-outputs
```

## Notes

- If generated files are missing or stale, re-run `dart run build_runner build --delete-conflicting-outputs`.
